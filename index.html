This page uses ES6 generators which only work in Firefox and Chrome so far. (To enable generators in Chrome, visit chrome://flags/#enable-javascript-harmony and click Enable.)

To use this page, hop into your developer tools (Cmd+Opt+I on a Mac in Chrome or Firefox) and type:

Dilemma.play(goodGuyGreg, scumbagSteve, 5)

You'll see console.log output for a 5-round game of Prisoner's Dilemma. goodGuyGreg cooperates every time, because that's how he is:

function* goodGuyGreg() {
    // Good Guy Greg always does the right thing.
    while (true)          // crashes on your couch
        yield COOPERATE;  // makes breakfast
}
